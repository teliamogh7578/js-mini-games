# JS Mini Games

A collection of mini games built using JavaScript while learning core concepts such as DOM manipulation, event handling, and basic game logic.  
This repository will continuously grow as new JavaScript concepts are learned and practiced.

## Purpose

The goal of this repository is to:

- Learn JavaScript through hands-on projects
- Apply each new concept by building a small interactive game
- Improve problem-solving and logical thinking
- Maintain a public record of consistent learning and progress

## Structure

Each game will be organized in its own folder with separate HTML, CSS, and JavaScript files.

## Learning Approach

- Every game is tied to one or more JavaScript concepts
- Games may start simple and be improved over time
- Code quality and structure will improve as learning progresses

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
